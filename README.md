# CCP

## Requirements

| Environment Variable | Description |
|----------------------|-------------|
| `SERVICE_ACCOUNT_SECRET` | The secret containing the service account credentials for accessing the Google Cloud Storage bucket. |
| `ROUNDCUBE_DEFAULT_USER_EMAIL` | The default email address for the Roundcube user. |
| `ROUNDCUBE_DEFAULT_USER_PASSWORD` | The default password for the Roundcube user. |
| `REALM_IMPORT_PATH` | The path to the realm import file for Keycloak. |
| `KeycloakAdminApiClientSecret` | The secret containing the credentials for accessing the Keycloak Admin API. |
